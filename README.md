# digia_engage_webengage

Monorepo for Digia Engage WebEngage plugins — bridges WebEngage in-app and personalization (inline) campaigns into Digia's server-driven UI rendering engine.

## Packages

- `android/` — Native Android library (`DigiaCEPPlugin` for WebEngage SDK + WE Personalization)
- `flutter/` — Flutter package (`WebEngagePlugin` implementing `DigiaCEPPlugin`)
